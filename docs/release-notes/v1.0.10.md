# v1.0.10

라이트/다크 컬러 위계 재매핑과 라이트모드 halation 보정. 토큰 이름·구조는 유지하고 값만 조정.

## 디자인 시스템 정비

- **라이트 컬러 위계 재매핑** — 다크 계층을 기준점으로 라이트에 인간 눈 보정(1.5~3배)을 적용해 1:1 매칭. 라이트 100% 근처에서 명도 차이가 둔감해지는 점 보정.
- **페이지/Chrome/Marquee 분리** — 라이트 본문 `#FFFFFF`, marquee `#F4F6F8`(-4%), 헤더·푸터 `#EBEEF2`(-6.5%). 다크의 본문 위 sunken/raised 구조와 의미 대응.
- **Border ramp 솔리드화** — alpha hex(`#XXXXXX15`)에서 솔리드로 전환해 단조성 회복(`faint < subtle < default < strong`). 이전엔 이름과 실제 강도가 역전돼 있었음.
- **다크 border 한 단계 약화** — `default` `#353A44` → `#292D35`로 부드럽게. bg 대비 17.4% → 11.4%로 자극 ↓.

## 컴포넌트 미세 조정

- **StatusBar** — 검색 pill·controls 그룹 배경 `transparent`화. hover는 배경이 아니라 글자/아이콘을 `sem.action.primary`로 변경(다크에서 라인과 충돌하던 hover 배경 제거). updateInfo 시간 텍스트도 동일 패턴.
- **시트 nav border** — `SheetLayout`, `StockDetailSheet`, `SearchSheet`, `StockDetailModal`, `WebViewPanel` 전체 `border.default` → `border.subtle`로 통일.
- **글로벌 랭킹** — 행 구분선 제거(원래 alpha 무효화로 보이지 않던 의도 복구). hover는 `action.primarySubtle`(accent 8% tint)로 약하게.
- **Tooltip** — 기본 `delay` 400ms → 200ms.

## 라이트모드 Halation 보정

- **font-smoothing 방향 정정** — 다크 `antialiased`(흰 글자 번짐 억제), 라이트 `auto`(subpixel = OS ClearType, 두께 살림). 이전엔 라이트에 `antialiased`가 적용돼 글자를 *더 얇게* 만들고 있었음.
- **text-shadow trick** — `body`에 `text-shadow: 0 0 0.5px currentColor` 적용. 명시적 font-weight 무관하게 모든 텍스트에 두께감 부여(inherit으로 전파).
- **무용 selector 제거** — emotion class를 잡지 못하던 `[style*="font-weight:..."]` selector 정리.
