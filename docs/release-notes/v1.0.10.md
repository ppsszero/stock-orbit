## v1.0.10 — 라이트모드 위계 재정비, 글자 가독성 보정

### 라이트모드 위계 다듬기
- 페이지·헤더/푸터·marquee 영역의 구분이 약하던 문제 해소 — 다크모드 기준 위계를 라이트에도 1:1 매핑(인간 눈이 다크에서 명도차에 더 민감해 라이트엔 더 강한 보정이 필요)
- 모달 안 취소 버튼이 모달 배경과 거의 같은 톤이라 묻혀 보이던 문제 — 카드 흰 배경 위에서 또렷하게 분리
- 모달 외곽 두꺼운 보더 약화 — 라인 의존도를 낮추고 그림자 중심으로 자연스럽게 떠있게

### 라이트모드 글자 가독성
- 라이트 텍스트가 다크보다 얇아 보이던 문제 보정 — halation 효과(사람 눈이 어두운 배경 위 흰 글자를 더 두껍게 인식하는 현상)를 글자 두께감으로 상쇄
- 모든 텍스트에 미세한 외곽 그림자(currentColor 기반)를 부여 — font-weight 명시 여부와 무관하게 일괄 적용

### 컴포넌트 디테일
- 하단 상태바 버튼 hover — 배경색 변경 대신 글자/아이콘이 강조색으로 변함 (다크모드에서 chrome 라인과 충돌하던 hover 배경 제거)
- 시트 상단 nav 아래 가로선 약화 — 시트 전반에 통일
- 글로벌 실시간 랭킹 행 구분선 제거 + 마우스 오버 톤 완화 (원래 디자인 의도 복구)
- 다크모드 보더/구분선 한 단계 완화 — 진하던 라인이 부드러워짐
- 툴팁 표시 속도 절반으로(400 → 200ms)

### 내부 정리
- 디자인 토큰 컬러값을 전면 재조정 (토큰 이름·구조는 그대로) — 라이트/다크 양쪽에서 의미·위계가 일치하도록
- alpha 기반 border 톤을 솔리드 hex로 전환 — 배경에 따라 사라지던 문제 해소, 이름과 실제 강도(faint < subtle < default < strong) 단조성 회복
